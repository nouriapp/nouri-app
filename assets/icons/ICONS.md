# NOURI — Iconography

See §5 of the root `README.md` for the full policy.

**TL;DR** — Linear, 1.5px stroke, rounded joins, no fill. Default set is
**Lucide** (CDN: `https://unpkg.com/lucide@latest`). No emoji ever. Color
inherits `currentColor`; action icons pick up `--nouri-purple`.

If NOURI ships with a custom icon set, drop `.svg` files into this folder
and reference them directly — `<img src="assets/icons/my-icon.svg" />` or
inline SVG.
